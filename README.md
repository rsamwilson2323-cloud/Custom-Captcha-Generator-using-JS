## 📌 Overview

A modern and interactive Captcha Generator that creates random captcha codes and validates user input in real time. Designed with a clean UI and smooth user experience to simulate basic human verification systems.

---

## ✨ Features

✅ Random Captcha Generation

✅ Refresh Captcha Button

✅ Real-Time Validation

✅ User-Friendly Interface

✅ Responsive Design

✅ Pure HTML, CSS & JavaScript

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
